# ROI Profit Calculator

Premium SaaS-style eCommerce ROI dashboard for eBay dropshippers and product researchers.

## Run locally

```bash
npm install
npm run dev
```

Then open `http://localhost:3000`.

## Included

- Next.js 15, React, Tailwind CSS, shadcn-style UI primitives, Lucide icons, and Recharts.
- Dashboard, Products, Calculator, History, Suppliers, Analytics, Settings, and Help views.
- Local storage database for products and settings.
- ROI, fee, margin, break-even, and profit calculations.
- CSV import, CSV export, Excel export, bulk delete, theme toggle, quick add, global search, product preview modal, responsive mobile navigation, toast notifications, and keyboard shortcuts.
